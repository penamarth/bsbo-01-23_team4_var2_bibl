# Словарь терминов

| Термин | Определение |
|--------|-------------|
| **Книга (Book)** | Экземпляр произведения, хранящийся в библиотеке, имеющий уникальный идентификатор и атрибут доступности. |
| **Читатель (Reader)** | Зарегистрированный пользователь библиотеки, имеющий право брать книги. |
| **Выдача (Loan)** | Запись, фиксирующая факт получения книги читателем, с датой выдачи и возможной датой возврата. |
| **Библиотекарь (Librarian)** | Пользователь системы, выполняющий учёт фонда, выдачу и возврат книг. |
| **Каталог (Catalog)** | Совокупность всех книг, зарегистрированных в библиотечной системе. |
| **Репозиторий (Repository)** | Программный компонент, управляющий доступом к данным (в памяти, базе или файле). |
| **UseCase (Сценарий использования)** | Завершённая пользовательская операция в рамках системы (например, "Выдать книгу"). |
| **Прецедент** | Конкретный случай использования системы, описанный в виде диаграммы последовательности. |
| **Доменная модель (Domain Model)** | Совокупность сущностей предметной области и связей между ними. |
| **Паттерн проектирования (Design Pattern)** | Типовое решение часто встречающейся проблемы проектирования программного обеспечения. |
| **Агрегация (Aggregation)** | Отношение "часть-целое" между объектами, где часть может существовать независимо от целого. |
| **Композиция (Composition)** | Строгая форма агрегации, где часть не может существовать без целого. |
| **Инкапсуляция (Encapsulation)** | Принцип ООП, объединяющий данные и методы работы с ними в единый объект. |
| **Интерфейс (Interface)** | Контракт, определяющий поведение класса без описания его реализации. |
| **Бизнес-правило (Business Rule)** | Ограничение или требование, определяющее работу предметной области. |
| **Сущность (Entity)** | Объект предметной области, имеющий уникальный идентификатор и жизненный цикл. |
| **Сервис (Service)** | Компонент системы, реализующий бизнес-логику, не относящуюся к конкретной сущности. |
| **Штраф (Fine)** | Денежное взыскание, начисляемое за просрочку возврата книги. |
| **Статус книги (Book Status)** | Текущее состояние книги: доступна, выдана, списана. |
| **Читательский билет (Library Card)** | Уникальный идентификатор читателя в библиотечной системе. |
