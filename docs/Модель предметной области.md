# Модель предметной области


```mermaid
erDiagram
    BOOK {
        string title
        string author
        bool isAvailable
    }
    
    READER {
        string name
        string libraryCard
    }
    
    LOAN {
        date loanDate
        date dueDate
        date returnDate
    }
    
    LIBRARY {
        string name
        string address
    }
    
    LIBRARIAN {
        string name
        string position
    }

    READER ||--o{ LOAN : "оформляет"
    BOOK ||--o{ LOAN : "выдана"
    LOAN ||--|| LIBRARIAN : "оформляет"
    BOOK }o--|| LIBRARY : "хранится в"
```

## Описание сущностей:
- **Книга (Book)** — основной объект учёта в библиотеке
- **Читатель (Reader)** — зарегистрированный пользователь библиотеки  
- **Выдача (Loan)** — факт передачи книги читателю на определенный срок
- **Библиотека (Library)** — организация, хранящая книжный фонд
- **Библиотекарь (Librarian)** — сотрудник, осуществляющий выдачу и прием книг
